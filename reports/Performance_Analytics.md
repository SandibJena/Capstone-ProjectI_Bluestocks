# Performance Analytics Report

## Objective

The objective of this task was to evaluate the performance of 40 mutual fund schemes using industry-standard financial metrics and benchmark comparisons.

---

## Dataset

- Historical NAV
- Benchmark Indices
- Fund Master
- Scheme Performance

---

## Performance Metrics Calculated

### Daily Returns

Calculated daily percentage returns for all 40 mutual fund schemes.

---

### CAGR

Computed

- 1-Year CAGR
- 3-Year CAGR
- 5-Year CAGR (where sufficient historical data was available)

---

### Sharpe Ratio

Measured risk-adjusted return using

Risk-Free Rate = 6.5%

---

### Sortino Ratio

Calculated downside risk-adjusted return using downside deviation.

---

### Alpha & Beta

Performed linear regression against the NIFTY100 benchmark.

Calculated

- Alpha
- Beta
- R²

---

### Maximum Drawdown

Calculated

- Running Maximum NAV
- Drawdown Series
- Maximum Drawdown
- Worst Drawdown Date

---

### Fund Scorecard

Built a composite score using

- 30% CAGR Rank
- 25% Sharpe Ratio
- 20% Alpha
- 15% Expense Ratio
- 10% Maximum Drawdown

---

### Benchmark Comparison

Compared the Top 5 Mutual Funds against

- NIFTY50
- NIFTY100

Calculated Tracking Error.

---

## Key Findings

- Long-term returns varied significantly across fund categories.
- Risk-adjusted metrics identified funds with consistent performance.
- Small-cap funds experienced higher drawdowns.
- Composite Fund Score provided an objective ranking framework.
- Benchmark comparison illustrated the relationship between fund performance and market indices.
